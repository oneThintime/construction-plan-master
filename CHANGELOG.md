# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [4.0.0] - 2026-03-17

### Added
- **Integrated System**: Merged single plan, multi-template, and phase-based generation modes
- **Interactive Menu**: User-friendly command-line interface
- **Complete Plan Generator**: Generate full construction plans with detailed content (8,000+ characters)
- **Desktop Shortcut**: Automatic desktop shortcut creation
- **Expert Review Tracking**: Automatic identification of 14 plans requiring expert review
- **Project Summary**: Master list generation for all 133 plans

### Features
- Single plan mode with Excel calculation sheets
- Phase-based batch generation (7 phases)
- AI prompt generation for content creation
- Emergency plan tracking (94 urgent plans)

### Project
- Complete BeiChe ShangGai School project (133 plans)
- 7 construction phases from preparation to handover

## [3.0.0] - 2026-03-17

### Added
- Phase-based generation system
- 133 plan configurations for BeiChe ShangGai School project
- Construction phase categorization (7 phases)
- Urgency level marking
- Phase summary generation

### Changed
- Restructured plan templates for project phases
- Enhanced output organization by phase

## [2.0.0] - 2026-03-17

### Added
- Multi-template system supporting 6 plan types:
  - Deep Excavation (深基坑)
  - High Formwork (高支模)
  - Scaffolding (脚手架)
  - Tower Crane Foundation (塔吊基础)
  - Temporary Electricity (施工用电)
  - Lifting Operations (起重吊装)
- Expert review threshold checking
- Configurable plan templates
- Extended calculation modules

### Features
- Template configuration system
- Plan type expansion capability
- Code and hazard database

## [1.0.0] - 2026-03-17

### Added
- Initial release of Construction Plan Master
- Basic single plan generation
- Excel calculation sheet generation (earth pressure, support force)
- AI content prompt generation
- Deep excavation plan templates
- Text-based plan outline generation

### Features
- Python-docx integration
- Openpyxl integration
- Basic CLI interface
