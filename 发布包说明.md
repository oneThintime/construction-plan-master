## 🚀 快速发布步骤

### Step 1: 创建 GitHub 仓库

1. 访问 https://github.com/new
2. 填写信息：
   - **Repository name**: `construction-plan-master`
   - **Description**: 施工方案编制大师 - A comprehensive construction engineering plan generation system for civil engineering projects
   - **Visibility**: Public
   - 勾选 ☑️ Add a README file
   - 勾选 ☑️ Add .gitignore: Python
   - 勾选 ☑️ Choose a license: MIT License
3. 点击 **Create repository**


## 🆘 常见问题

### Q: 推送失败怎么办？
```bash
# 强制推送
git push -u origin main --force

# 或先拉取再推送
git pull origin main --allow-unrelated-histories
git push -u origin main
```

### Q: 如何更新已发布的版本？
```bash
# 修改代码后
git add .
git commit -m "fix: bug fixes"
git push

# 然后在 GitHub 创建新 Release
tag: v4.0.1
title: v4.0.1 Bug Fixes
```

### Q: 别人如何安装使用？
```bash
# 通过 pip 安装（发布后）
pip install construction-plan-master

# 或通过 git 克隆
git clone https://github.com/yourusername/construction-plan-master.git
cd construction-plan-master
pip install -r requirements.txt
python construction_plan_master.py
```
