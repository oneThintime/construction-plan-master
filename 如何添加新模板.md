# 如何添加新的方案模板

本指南说明如何为施工方案编制系统添加新的方案类型模板。

---

## 📁 模板配置文件

所有方案模板定义在：
```
templates\plan_configs.py
```

---

## 📝 添加新模板的步骤

### 步骤 1：在 PLAN_TEMPLATES 中添加配置

打开 `templates\plan_configs.py`，在 `PLAN_TEMPLATES` 字典中添加新方案：

```python
"新方案类型": {
    "description": "方案描述",
    "expert_threshold": {"param": "关键参数名", "value": 阈值数值, "unit": "单位"},
    "codes": [
        "《规范名称》规范编号",
        "《另一个规范》编号"
    ],
    "hazards": ["危险源1", "危险源2", "危险源3"],
    "calculations": ["calc_type_1", "calc_type_2"],
    "required_params": ["param1", "param2"],
    "ai_sections": ["工程概况", "编制依据", "施工工艺"]
},
```

### 步骤 2：配置字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| `description` | 方案类型的描述 | `"高大模板支撑系统施工方案"` |
| `expert_threshold` | 专家论证阈值 | `{"param": "height", "value": 8, "unit": "m"}` |
| `codes` | 编制依据（规范列表） | `["JGJ 162-2008", ...]` |
| `hazards` | 危险源识别列表 | `["坍塌", "高处坠落", ...]` |
| `calculations` | 计算书类型标识 | `["scaffold_stability", "load_calculation"]` |
| `required_params` | 必需参数列表 | `["height", "span", "load"]` |
| `ai_sections` | AI生成内容章节 | `["工程概况", "施工工艺"]` |

### 步骤 3：（可选）添加专用计算

如果新方案需要特殊计算，在 `templates\calculations.py` 中添加：

```python
def generate_your_calculation(self, params):
    """你的计算书生成方法"""
    ws = self.wb.create_sheet(title="计算表名称")
    
    # 标题
    ws.merge_cells('A1:D1')
    ws['A1'] = "计算书标题"
    ws['A1'].font = Font(bold=True, size=16)
    
    # 参数和计算...
    
    return self.wb
```

然后在 `plan_generator_v2.py` 的 `generate_calculations` 方法中调用：

```python
if plan_type == '你的新类型':
    self.xlsx_gen.generate_your_calculation(project_info)
```

## 💡 示例：添加 "钢结构安装" 方案

### 1. 添加模板配置

```python
"钢结构安装": {
    "description": "建筑钢结构安装工程施工方案",
    "expert_threshold": {"param": "weight", "value": 100, "unit": "kN"},
    "codes": [
        "《钢结构工程施工质量验收规范》GB 50205-2020",
        "《钢结构焊接规范》GB 50661-2011",
        "《钢结构高强度螺栓连接技术规程》JGJ 82-2011"
    ],
    "hazards": ["坍塌", "高处坠落", "起重伤害", "火灾", "触电", "物体打击"],
    "calculations": ["lifting_capacity", "connection_check", "stability_check"],
    "required_params": ["weight", "lift_height", "crane_model"],
    "ai_sections": ["工程概况", "编制依据", "安装工艺", "焊接工艺", "安全措施"]
},
```

### 2. 添加计算书方法（可选）

```python
def generate_steel_structure_calc(self, params):
    """钢结构安装计算书"""
    ws = self.wb.create_sheet(title="钢结构计算")
    
    ws.merge_cells('A1:D1')
    ws['A1'] = "钢结构安装计算书"
    ws['A1'].font = Font(bold=True, size=16)
    
    # 构件信息
    ws['A3'] = "构件参数"
    ws['A3'].font = Font(bold=True, size=12)
    
    ws['A4'] = "构件重量 (kN)"
    ws['B4'] = params.get('weight', 100)
    ws['A5'] = "安装高度 (m)"
    ws['B5'] = params.get('lift_height', 30)
    ws['A6'] = "吊车型号"
    ws['B6'] = params.get('crane_model', 'QY100')
    
    # 吊装验算...
    
    return self.wb
```

### 3. 在生成器中调用

```python
if plan_type == '钢结构安装':
    steel_params = {
        'weight': project_info.get('weight', 100),
        'lift_height': project_info.get('lift_height', 30),
        'crane_model': project_info.get('crane_model', 'QY100')
    }
    self.xlsx_gen.generate_steel_structure_calc(steel_params)
```

### 4. 添加命令行参数

在 `plan_generator_v2.py` 的 `main()` 函数中添加参数：

```python
parser.add_argument('--crane-model', default='QY100', help='吊车型号')
parser.add_argument('--lift-height', type=float, default=30, help='安装高度(m)')
```

## 📋 现有模板参考

目前系统支持以下方案类型：

| 方案类型 | 专家论证阈值 | 主要计算 |
|---------|-------------|---------|
| 深基坑 | depth ≥ 5m | 土压力、支撑轴力 |
| 高支模 | height ≥ 8m | 支架稳定性、荷载 |
| 脚手架 | height ≥ 50m | 稳定性、风荷载 |
| 塔吊基础 | 按型号 | 地基承载力、稳定性 |
| 施工用电 | 无 | 荷载计算、电缆选型 |
| 起重吊装 | weight ≥ 100kN | 起重能力、吊索具 |

## 🆘 常见问题

**Q: 添加模板后报错 "KeyError"？**  
A: 检查 `plan_configs.py` 中的字段名是否拼写正确。

**Q: 新模板的计算书没有生成？**  
A: 确保在 `generate_calculations` 方法中添加了对应的判断和调用。

**Q: 如何设置不需要专家论证的方案？**  
A: 将 `expert_threshold` 设为 `None`。
