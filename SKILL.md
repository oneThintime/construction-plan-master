# 施工方案编制大师 V4.0 - SKILL.md

施工方案编制大师（Construction Plan Master）- 专为北车上盖学校项目设计的综合方案编制系统。

## 版本历史

| 版本 | 文件 | 功能 |
|------|------|------|
| V1.0 | construction_plan_enhanced.py | 基础单方案生成 |
| V2.0 | plan_generator_v2.py | 多模板支持（6种方案） |
| V3.0 | phase_generator_v3.py | 按阶段批量生成（133方案） |
| **V4.0** | **construction_plan_master.py** | **综合版：单方案+多模板+按阶段** |

---

## 功能特点

### 1. 三种工作模式

| 模式 | 用途 | 输出 |
|------|------|------|
| **单方案生成** | 生成单个危大工程方案 | 方案大纲+Excel计算书+AI提示 |
| **按阶段生成** | 批量生成指定阶段方案 | 阶段目录+汇总清单 |
| **全部生成** | 一次性生成133个方案 | 7个阶段+项目总清单 |

### 2. 施工阶段划分（北车上盖学校项目）

```
阶段1_施工准备          17个方案  (紧急13个)
阶段2_桩基基坑          14个方案  (紧急10个, 需论证2个)
阶段3_基础施工          14个方案  (紧急11个, 需论证3个)
阶段4_主体结构          32个方案  (紧急23个, 需论证5个)
阶段5_装饰装修          17个方案  (紧急16个, 需论证3个)
阶段6_机电安装          16个方案  (紧急9个, 需论证1个)
阶段7_室外及收尾        23个方案  (紧急12个)
────────────────────────────────────────────
总计: 133个方案
```

### 3. 方案级别分类

| 级别 | 类型 | 数量 | 说明 |
|------|------|------|------|
| A类 | 超过一定规模的危大工程 | 1个 | 需专家论证 |
| B类 | 危大工程 | 29个 | 需编制专项方案 |
| C类 | 一般安全专项 | 6个 | 需审批 |
| D类 | 普通施工方案 | 80个 | 项目部审批 |

## 使用方法

### 方式 1：交互式菜单（推荐）

```
双击运行: 方案大师V4.bat
```

菜单选项：
- [1] 单方案生成 - 输入工程参数，生成单个方案
- [2] 按阶段生成 - 选择阶段(1-7)，批量生成
- [3] 查看阶段列表 - 显示所有阶段信息
- [4] 生成所有阶段 - 一次性生成133个方案
- [0] 退出

### 方式 2：命令行

```bash
cd C:\Users\18771\.openclaw\skills\construction-plan-enhanced

# 单方案生成
python construction_plan_master.py --mode single --name "测试工程" --type "深基坑" --depth 12

# 查看阶段列表
python construction_plan_master.py --mode list

# 生成指定阶段（如阶段1）
python construction_plan_master.py --mode phase --phase 1

# 生成所有阶段
python construction_plan_master.py --mode all --name "北车上盖学校"
```

## 配置文件

### 施工阶段配置
文件: `templates/phase_configs.py`

包含7个施工阶段的133个方案配置：
- 方案名称
- 方案类别
- 方案级别（A/B/C/D）
- 紧急程度
- 是否需要专家论证

### 方案模板配置
文件: `templates/plan_configs.py`

包含6种通用方案类型的模板配置：
- 深基坑
- 高支模
- 脚手架
- 塔吊基础
- 施工用电
- 起重吊装

## 如何添加新方案

### 添加单个方案到现有阶段

编辑 `templates/phase_configs.py`，在对应阶段的 `plans` 列表中添加：

```python
{
    "name": "新方案名称",
    "type": "方案类别",
    "level": "D",  # A/B/C/D
    "urgent": False,  # 是否紧急
    "expert_required": False  # 是否需要专家论证
}
```

### 添加新方案类别

编辑 `templates/phase_configs.py`，在 `_get_sections_by_type` 方法中添加章节结构：

```python
"新类别": [
    "编制依据",
    "工程概况",
    "施工工艺",
    "安全措施",
    "质量标准"
],
```
